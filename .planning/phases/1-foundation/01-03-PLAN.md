1. Implement `Parapet.Internal.SafeHandler`. Define `def attach(handler_id, event_name, handler_module, function_name, config \\ %{})`.
It must call `:telemetry.attach/4`. The inline callback function must wrap `apply(handler_module, function_name, ...)` in a `try/rescue` block.
On rescue, it must capture the stacktrace using `__STACKTRACE__` and call `Logger.error("Parapet telemetry handler exception in #{inspect(handler_module)}.#{function_name}/4", event: event, exception: Exception.message(e), stacktrace: inspect(__STACKTRACE__))`.

2. Implement `Parapet` (in `lib/parapet.ex`) with a `@moduledoc` explaining the boundary constraints. Define `def attach(args)`.
It delegates attachments through `Parapet.Internal.SafeHandler` and returns `{:ok, handlers}` or `{:error, reason}` (where `handlers` is the list or identifier of attached handlers).
It should also contain `Logger.debug("Parapet attached telemetry handler #{handler_id}")` upon success.
If an optional dependency is missing (simulated by a `Code.ensure_loaded?` wrap), emit a warning via `Logger.warning`.
