  <action>
    Create `Parapet.Notifier.Slack` module and declare it as an implementation of `@behaviour Parapet.Notifier`.
    Implement the `deliver(incident, opts)` callback. Build a rich Slack Block Kit JSON payload using the data from the `incident` struct (such as incident ID, status, and title). 
    CRITICAL: You must include a "direct link to the Parapet Operator UI" in the Block Kit payload. Use a configurable base URL from `opts[:operator_url]` or similar to construct the link to the specific incident.
    Extract the Slack Incoming Webhook URL from `opts[:webhook_url]`.
    Use `Req.post(url, json: payload)` to send the payload to Slack.
    Handle the HTTP response from `Req`. Return `{:ok, response_body}` on a 2xx status code, and `{:error, reason}` on HTTP errors or non-2xx status codes.
    Write tests verifying the Block Kit payload structure, specifically asserting the presence of the UI link.
