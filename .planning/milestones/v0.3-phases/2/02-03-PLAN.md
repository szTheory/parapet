  - [ ] **Secure Module Loading:** Inside `execute_runbook_step`, extract the `module` string from `incident.runbook_data`. Use `String.to_existing_atom` wrapped in a `try/rescue` ArgumentError block to safely convert to a module atom (preventing atom exhaustion).
  - [ ] **Validate Function Exists:** Convert `step_id` to an existing atom. Check `function_exported?(module, :execute_mitigation, 2)` before applying to prevent crashes on obsolete runbooks.
  - [ ] **Execute Callback:** Call `apply(module, :execute_mitigation, [step_atom, incident])`.
  - [ ] **Record Audit & Evidence:** Wrap the result into `timeline_attrs` and `audit_attrs` (using `build_audit`), and pass them to `Evidence.run_operator_command/1` to commit the transaction and preserve immutable facts.

  - [ ] **Write Tests:** Add test cases in `test/parapet/operator_test.exs` to verify successful execution records a ToolAudit.
