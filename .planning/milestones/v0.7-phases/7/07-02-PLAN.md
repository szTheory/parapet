## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Operator UI intent -> host-wired recovery execution | Human-triggered recovery crosses from evidence review into mutation |
| Preview payload -> confirm execution | Reviewed scope must not drift between preview and execute |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-02-01 | Tampering | preview to confirm seam | mitigate | Require preview token or snapshot match and fail closed on stale or mismatched scope. |
| T-07-02-02 | Repudiation | recovery execution | mitigate | Record both preview and confirm through timeline or audit with actor, reason, capability, scope, and outcome. |
| T-07-02-03 | Elevation of Privilege | optional capabilities | mitigate | Execute only when the host has explicitly registered the named capability. |
</threat_model>
